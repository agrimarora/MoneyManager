# 💰 MoneyManager - Personal Finance Tracker

**MoneyManager** is a modern, feature-rich Android application built with Jetpack Compose. It empowers users to take control of their finances by tracking expenses, setting savings goals, and staying informed with the latest financial news.

![App Logo](app/src/main/res/drawable/app_logo.png)

---

## 🌟 Key Features

- **📊 Comprehensive Dashboard**: Get a quick overview of your total balance, income, and expenses with beautiful interactive charts.
- **💸 Expense Tracking**: Easily record your daily transactions with categories and descriptions.
- **🎯 Savings Goals**: Define and track your progress toward financial milestones.
- **📈 Financial Analytics**: Detailed reports and history to visualize your spending patterns over time.
- **📰 Financial News**: Stay updated with real-time news related to the financial world.
- **🤖 AI Assistant**: Integrated ChatBot powered by Gemini for smart financial advice and queries.
- **🔐 Secure Authentication**: Firebase-backed login and sign-up with support for Google Identity.
- **🌗 Material 3 Design**: Fully responsive and adaptive UI with Dark Mode support.

---

## 🛠️ Tech Stack

- **Lanuage**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/compose)
- **Architecture**: MVVM (Model-View-ViewModel) with Clean Architecture principles.
- **Dependency Injection**: [Hilt](https://dagger.dev/hilt/)
- **Networking**: [Retrofit](https://square.github.io/retrofit/) & [OkHttp](https://square.github.io/okhttp/)
- **Database**: [Room](https://developer.android.com/training/data-storage/room) & [Firebase Firestore](https://firebase.google.com/docs/firestore)
- **Auth**: [Firebase Auth](https://firebase.google.com/docs/auth) & [Google Identity](https://developers.google.com/identity/sign-in/android)
- **Charts**: [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart) & [YCharts](https://github.com/yml-org/ycharts)
- **AI Integration**: [Google AI SDK (Gemini)](https://ai.google.dev/)
- **Image Loading**: [Coil](https://coil-kt.github.io/coil/)

## 🚀 Getting Started

### Prerequisites

- Android Studio Koala or newer.
- JDK 11 or higher.
- A Firebase project with `google-services.json` placed in the `app/` directory.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/MoneyManager.git
   ```
2. **Open in Android Studio**:
   Select the `MoneyManager` directory and let Gradle sync.
3. **Run the app**:
   Connect an Android device or use an emulator and click the **Run** button.
